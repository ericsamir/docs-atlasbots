O ranking de economia é uma das ferramentas mais poderosas para criar competição saudável no servidor. Ao exibir quem acumulou mais moedas, você incentiva os membros a participarem mais das conversas, resgatarem suas recompensas diárias e se envolverem com todas as mecânicas da economia — tudo para subir na classificação e mostrar quem manda por ali.

## Usando o ranking

Para consultar o ranking, use os comandos abaixo diretamente no chat do Discord:

| Comando | Descrição |
|---|---|
| `/ranking` | Exibe o top 10 dos membros com maior saldo no servidor |
| `/ranking [pagina]` | Navega pelas páginas do ranking (ex: `/ranking 2` para a segunda página) |
| `/saldo [usuário]` | Verifica o saldo de um membro específico do servidor |

## Administrando saldos

Como administrador do servidor, você tem controle total sobre os saldos dos membros. Use os comandos abaixo com permissão de administrador:

| Comando | Descrição |
|---|---|
| `/economia dar [usuário] [valor]` | Adiciona moedas ao saldo de um membro |
| `/economia remover [usuário] [valor]` | Remove moedas do saldo de um membro |
| `/economia definir [usuário] [valor]` | Define um valor exato de saldo para um membro |
| `/economia resetar [usuário]` | Zera o saldo de um membro específico |
| `/economia resetar-todos` | **Apaga o saldo de todos os membros do servidor** |

<Warning>
  O comando `/economia resetar-todos` é **irreversível**. Depois de executado, todos os saldos do servidor serão zerados permanentemente e não há como desfazer essa ação. Use com extrema cautela e, se possível, avise os membros com antecedência.
</Warning>

<Tip>
  Que tal organizar **eventos especiais** com recompensas em moedas? Você pode usar o `/economia dar` para premiar vencedores de sorteios, concursos de memes, giveaways ou desafios da comunidade. Esse tipo de evento movimenta o ranking e deixa o servidor muito mais animado.
</Tip>
