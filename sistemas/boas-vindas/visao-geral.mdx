O sistema de boas-vindas do Atlas Bots transforma a primeira impressão do seu servidor em algo memorável. Quando um novo membro entra, o bot envia automaticamente uma mensagem personalizada no canal escolhido — com texto, embed colorido e até imagem — para que todos saibam da chegada. Você também pode configurar uma mensagem de despedida para quando alguém sair, além de uma DM privada de boas-vindas diretamente para o novo membro.

## Principais recursos

Confira tudo o que o sistema de boas-vindas oferece para o seu servidor:

| Recurso                  | Descrição                                                                 |
| ------------------------ | ------------------------------------------------------------------------- |
| **Mensagem de entrada**  | Envia uma mensagem personalizada quando um novo membro entra no servidor. |
| **Mensagem de saída**    | Notifica o canal quando um membro deixa o servidor.                       |
| **DM de boas-vindas**    | Envia uma mensagem privada diretamente ao novo membro ao entrar.          |
| **Imagem de boas-vindas**| Gera uma imagem personalizada com avatar e nome do novo membro.           |
| **Cargos automáticos**   | Atribui cargos predefinidos automaticamente a todo novo membro.           |

<Note>
  Para que o sistema de boas-vindas funcione corretamente, é obrigatório ter ao menos um **canal configurado** como destino das mensagens. Sem isso, as notificações não serão enviadas, mesmo que o sistema esteja ativado.
</Note>
